Taxonomy
Recommended classification — 11 facets — aligned with Figma designs & existing canonical taxonomy
Expand All / Collapse All
1. Canonical Entities▼
  • Coverage — Insurance plan record (dental, vision, life, disability, supplemental, EAP)
  • Claim — A submitted request for benefits payment
  • Leave / Absence — A requested period away from work
  • Payment — Financial transaction tied to a claim
  • Document — PDF/form artifact (EOB, plan docs, tax forms)
  • Provider — Healthcare professional or facility
  • Message — Async communication between member and support
  • Support Resource — FAQ, article, or help content
  • ID Card — Digital insurance identification card (NEW)
  • Notification — System-generated alert or task (NEW)
2. Benefit Types▼
  • Dental — Premier PPO Dental Plan
  • Vision — Premier PPO Vision Plan
  • Life Insurance — Basic — Employer-provided
  • Life Insurance — Voluntary — Employee-elected
  • Disability — Short Term — 60% salary, 24 weeks max
  • Disability — Long Term — 60% salary, to age 65
  • Accident Insurance
  • AD&D
  • Hospital Indemnity
  • Critical Illness
  • Employee Assistance Program (EAP)
3. Coverage Attributes▼
  • Coverage Type — Core — Employer-provided base benefits
  • Coverage Type — Voluntary — Employee-elected, employee-paid
  • Coverage Type — Supplemental — Add-on protection
  • Coverage Level — Employee Only | You + Family
  • Network Status — In-Network | Out-of-Network (NEW)
  • Plan Year — Annual filter on coverages list
  • Has Claim — Yes | No
4. User Action Taxonomy▼
  • Understand Coverage — View plan details, compare benefits
  • Start a Claim — Submit new claim (member or guest)
  • Track a Claim or EOB — View claim status, download EOB
  • Request Absence — Multi-step intake wizard
  • Plan Absence — Pre-request planner
  • My Absence — View active/past leaves, filter, resume drafts
  • Update Absence — Modify dates, report missed time (NEW)
  • Return to Work — Confirm date, schedule, clearance (NEW)
  • Get Help or Support — Search FAQ, contact support
  • View ID Card — Find, print, download, share (NEW)
  • Find a Provider — Search dental/vision providers (NEW)
  • Estimate Cost — Search procedures, view estimates (NEW)
  • Manage Profile — Update info, security, preferences (NEW)
  • Manage Dependents — Invite, view, change access (NEW)
  • View Documents — Browse/download plan documents (NEW)
5. System Activity Taxonomy▼
  • Alert — Urgent — Requires immediate action
  • Alert — Informational — FYI updates
  • Notification — System event
  • Message — Async support communication
  • Task — Action required by member (NEW)
6. User Roles & Access▼
  • Subscriber (Primary) — Full access to all data (EAC)
  • Dependent — Limited access, age-restricted
  • Self-Administered Employee — Little to no data
  • Partial Roster Employee — Moderate access
  • Guest / Unauthenticated — Claim submission only
7. Authentication State▼
  • Unauthenticated — Login page, guest claim
  • Registration — Invite — Employer email invite flow
  • Registration — Self — Self-service via link
  • Registration — Dependent — Invited by primary subscriber
  • MFA Challenge — Identity verification
  • Authenticated — Full portal access
8. Absence / Leave Taxonomy▼
  • Absence Reason — Illness/Injury | Family | Non-Birthing | Birthing | Military | Other
  • Leave Frequency — Continuous | Intermittent | Reduced Schedule
  • Absence Status — Draft | Pending | Approved | Denied | Completed
  • Benefits Applied — FMLA | STD | State PFL | Bonding | Unpaid
  • Absence Lifecycle — Plan → Request → Review → Approved → On Leave → Update → RTW → Completed
  • Return to Work — Date | Schedule | Clearance | Accommodations
  • Documentation — FMLA Cert | Medical | Provider Statement | Employer Verification
9. Content / Document Types▼
  • EOB — Claim payment summary
  • Plan Document — Coverage terms and details
  • Tax Form — 1095-B and related
  • Claim Form — Submission forms
  • ID Card — Digital card (front/back)
  • FAQ Article — Help content
  • Resource Article — Educational content by benefit type
  • Notification — Alert/task card
10. Support Channel▼
  • Message — Asynchronous text support
  • Live Chat — Real-time chat
  • Phone — Department-specific numbers
  • Self-Service — FAQ, articles, resources
11. Platform / Device▼
  • Desktop Web — Full portal (Benefits Hub nav, multi-column layout)
  • Mobile Web — Responsive portal (hamburger nav, stacked layout)
  • Email — Invite/notification emails (InsurePortal branded)
